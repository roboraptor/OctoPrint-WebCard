# OctoPrint-WebCard

This plugin for OctoPrint adds a tab to the UI that displays a configurable webpage.

## Features

*   Display any webpage in a tab in the OctoPrint UI.
*   The URL of the webpage is configurable in the settings.
*   You can choose to display a title or a FontAwesome icon for the tab.

## Setup

Install via the bundled Plugin Manager or manually using this URL:

```
https://github.com/roboraptor/OctoPrint-WebCard/archive/main.zip
```
